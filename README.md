# StockDaddy

**StockDaddy** is a C# inventory management application for tracking products, stock levels, and inventory activity in one place.

## Overview

StockDaddy helps you:

- Keep a centralized list of inventory items
- Track quantities on hand
- Monitor low-stock items
- Record inventory updates (additions, removals, corrections)
- Improve visibility into day-to-day stock operations

## Features

- 📦 Product catalog management
- 🔢 Quantity and stock-level tracking
- ⚠️ Low-stock monitoring
- 🧾 Inventory transaction/history logging
- 🔍 Search and filter inventory items
- 🛠️ Built with C#
